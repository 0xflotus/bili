# Installation

Bili requires [Node.js](https://nodejs.org/en/) 8 or above:

```bash
# Check your Node.js version
node -v
```

Install Bili locally as dev dependency using npm:

```bash
cd your-project
npm i bili -D
```

You can also install it with [Yarn](https://yarnpkg.com/) if you want:

```bash
cd your-project
yarn add bili --dev
```
